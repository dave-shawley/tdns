tdns
====
An asynchronous Tornado `pycares <http://pycares.readthedocs.io>`_ DNS
client wrapper, exporting the full API.

|Version| |Downloads| |License| |PythonVersions|

Installation
------------
tdns is available from the `Python Package Index <https://preview-pypi.python.org/project/tdns/>`_ and can be installed by running :command:`pip install tdns`
